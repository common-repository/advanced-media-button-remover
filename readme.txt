This plugin removes selected media buttons from visual post editor.

== Description ==

There are cases where you don't need the full set of media upload buttons in the visual post editor. This plugin lets you remove the buttons individually. The plugin is configured through an own configuration screen (see screenshot)**.**

**Please note:** Buttons(and its functionality) are not entirely removed, but only hidden.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
